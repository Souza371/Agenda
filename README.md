# Agenda
Sistema Modular de Agendamento de Consultas para Clínicas
Este projeto é composto por três principais módulos:

backend/ — API RESTful, autenticação, notificações, banco de dados
frontend/ — Aplicação web responsiva (React.js)
mobile/ — Aplicativo mobile (React Native)
Funcionalidades
Cadastro de clínicas, profissionais e pacientes
Agendamento, edição e cancelamento de consultas
Visualização de agenda
Notificações automáticas (e-mail, SMS, WhatsApp)
Personalização visual
Controle de permissões
Multiusuário e multiplataforma
Tecnologias
Node.js/Express, PostgreSQL, React.js, React Native, JWT, Swagger, Twilio, Gmail API
